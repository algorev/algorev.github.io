La Ville des Chats

Les chats de la ville ont disparu. Sauf pour quelques uns qui sont restés, nul ne sait où ils sont passés. Vous n'aviez qu'eux. Pas de famille, pas d'amis, le bol de lait que vous leur laissiez le soir sur votre fenêtre était la seule marque de votre présence au monde. Une nuit, un chat vient vous chercher dans votre chambre: vous êtes la dame au lait. Vous devez venir l'aider. Les chats de la ville sont sur le point de partir, et vous devez les convaincre de rester.

Si vous n'êtes pas familier.e avec la fiction interactive, vous pouvez jeter un coup d'oeil aux Instructions.

Si à un moment dans le jeu, vous ne savez pas quoi faire/êtes coincé.e, vous pouvez regarder le Guide.

Pour plus d'informations sur les outils et les ressources qui ont été utilisés dans le jeu, regardez les Crédits du jeu.
